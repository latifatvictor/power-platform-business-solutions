# ✈️ International Travel Request & Approval System

## 📌 Overview

A Power Apps and Power Automate solution designed to streamline and digitise the process of submitting, approving, and tracking international travel requests.

This solution replaces manual, email-based processes with a structured and automated workflow, improving efficiency, visibility, and accountability.

---

## 🎯 Objectives

- Eliminate manual travel request processes  
- Standardise approval workflows  
- Improve visibility of request status  
- Reduce delays in approvals and coordination  
- Enhance tracking and reporting of travel activities  

---

## ⚙️ Key Features

- User-friendly travel request submission form  
- Multi-stage approval workflow (e.g. Manager → Finance → HR)  
- Automated email notifications and updates  
- Real-time request status tracking  
- Centralised data storage using SharePoint  
- Audit trail for all requests and approvals  

---

## 🛠️ Technologies Used

- Power Apps (Canvas App)  
- Power Automate (Flow automation)  
- SharePoint Online (data storage)  
- Microsoft 365  

---

## 🔄 Process Flow

1. User submits a travel request via Power Apps  
2. Request is stored in SharePoint  
3. Approval workflow is triggered via Power Automate  
4. Approvers receive automated notifications  
5. Approvals/rejections are recorded  
6. User receives status updates  
7. Final outcome is logged and stored  

---

## 📈 Business Impact

- Reduced reliance on email-based approvals  
- Improved turnaround time for request processing  
- Increased transparency and visibility across stakeholders  
- Enhanced accountability with audit tracking  
- Standardised travel request processes across teams
