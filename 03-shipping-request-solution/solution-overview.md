# Solution Overview
## Logistics & Shipping Request Automation Solution

This solution is a Power Apps and Power Automate application designed to streamline the submission, processing, and tracking of shipping requests.

### Core components
- Power Apps for request submission and tracking  
- SharePoint for data storage  
- Power Automate for workflow automation and notifications  

### Key benefits
- standardised shipping request process  
- improved tracking and visibility  
- reduced manual coordination  
- faster request processing  

### Workflow summary
1. User submits shipping request  
2. Request is stored in SharePoint  
3. Approval (if required) is triggered  
4. Logistics team processes the request  
5. Status is updated and communicated  

This solution demonstrates how Power Platform can be used to improve operational workflows and logistics coordination.
